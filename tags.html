Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Document Tags

Tag Purpose Required for valid HTML 5
html defines an HTML document yes
head non-displayed document data yes
body the content itself no
title title for the document yes (non-empty)
meta additional data relating to the document no

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header mark out introductory content
div division split content into logical sections
ol ordered list create numbered list of elements
ul unordered list create bullet point list of elements
li list element individual elements within ol or ul
blockquote blockquote format for a quotation

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized test <em>techinical sophistication</em> techinical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code include a block of code <code>sudo rm -fr /</code> sudo rm -fr /
span span mark a section of text <span>This text</span> This text
br line break insert line break Line<br>Break Line
